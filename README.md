# ToolBox Online

A collection of free, fast, and privacy-friendly online tools. No sign-up required — everything runs directly in your browser.

## Tools

| Tool | Description |
|------|-------------|
| QR Code Generator | Generate QR codes from text, URLs, phone numbers |
| Word & Character Counter | Count words, characters, sentences, reading time |
| Password Generator | Create strong, random passwords with custom options |
| Case Converter | Convert text to UPPERCASE, lowercase, Title Case, camelCase, snake_case... |
| Color Converter | Convert between HEX, RGB, HSL with a visual color picker |
| JSON Formatter | Format, minify, and validate JSON |
| Base64 Encode/Decode | Encode and decode Base64 with Unicode support |
| Hash Generator | Generate MD5, SHA-1, SHA-256, SHA-512, SHA-3 hashes |
| Lorem Ipsum Generator | Generate placeholder text for design and development |
| Markdown to HTML | Convert Markdown to HTML with live preview |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Next.js** — React framework with SSR/SSG
- **TypeScript** — Type safety
- **Tailwind CSS** — Utility-first CSS
- **QRCode.js** — QR code generation
- **CryptoJS** — Hash generation

## Key Features

- **Fast & Free** — All tools run entirely in the browser, no server processing
- **Privacy First** — No data is stored or sent to any server
- **Responsive** — Works on desktop, tablet, and mobile
- **SEO Optimized** — Auto-generated sitemap and meta tags
